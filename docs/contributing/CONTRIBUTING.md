# Как вносить вклад в проект

## Ветки
- `main` — только стабильные версии (защищена)
- `develop` — основная ветка разработки
- `feature/*` — ветки для новых задач (например, `feature/core-hello`)

## Коммиты
Пишем осмысленные сообщения на английском:
- `feat: add player movement` — новая функция
- `fix: correct health bar position` — исправление бага
- `docs: update README` — документация

## Pull Requests
1. Создаёте ветку от `develop`
2. Выполняете задачу
3. Открываете PR в `develop`
4. В описании PR указываете: "Closes #номер_issue"
5. Ждёте ревью (минимум 1 одобрение)
6. Ментор мёржит

## Code Review
- Будьте вежливы
- Проверяйте, что код работает
- Ищите потенциальные проблемы
- Предлагайте улучшения
